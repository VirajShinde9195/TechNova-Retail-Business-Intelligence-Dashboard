# 📊 TechNova Retail Business Intelligence Dashboard

## 📌 Project Overview

This project is an interactive Business Intelligence Dashboard developed using Microsoft Power BI for **TechNova Retail Pvt. Ltd.**

The objective of the project is to analyze retail sales performance, profitability, customer behavior, regional performance, and business trends to support data-driven decision making.

---

# 🎯 Business Problem

The management observed that although revenue was increasing, profitability was declining in certain regions.

They wanted answers to the following business questions:

- Which products generate the highest profit?
- Which products are causing losses?
- Which customers contribute the most revenue?
- Are discounts affecting profitability?
- Which cities and states perform well?
- Which sales executives require improvement?
- Which product categories deserve more investment?
- How does seasonality affect sales?

# 📂 Project Workflow

## 1. Data Collection

- Imported CSV dataset into Power BI.

---

## 2. Data Cleaning (Power Query)

Performed the following preprocessing steps:

- Removed duplicate records
- Handled missing values
- Replaced null Customer Ratings with Average Rating
- Checked data types
- Removed data inconsistencies
- Applied transformations
- Loaded cleaned data into Power BI

---

## 3. Data Modeling

Created:

- Date Table
- Measures Table

Established relationships for efficient filtering and time intelligence.

---

## 4. DAX Measures

Created multiple business KPIs including:

- Total Sales
- Total Profit
- Total Cost
- Total Orders
- Total Customers
- Total Quantity
- Average Order Value
- Average Discount
- Average Rating
- Average Delivery Days
- Profit Margin %
- Returned Orders
- Return Rate %
- Average Profit Per Order
